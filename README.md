# Cvefinder
A simple commandline app for searching and looking up opensource vulnerabilities

# Installation

## Using Pip
```bash
  $ pip install cvefinder
```

# Usage
```bash
$ cvefinder
```
## Search
`search <keyword>`
```bash
$ cvefinder search python
```
## Lookup
`search <name>`
```bash
$ cvefinder look-up CVE-2020-2121
```
